## ✨ 主な機能 (Features)

### 📄 PDF・画像編集
- **PDF読み込み**: ドラッグ＆ドロップでPDFファイルを即座にプレビュー・編集。
- **画像挿入**: PNG/JPEG画像をページとして挿入可能。
- **ページの並べ替え**: サイドバーでページを直感的にドラッグ＆ドロップ。
- **ページの操作**: 回転、複製、削除、一括選択（Ctrl+Click）。

### ✏️ 注釈・マークアップ
- **テキスト追加**: 自由な位置にテキストを追加。フォントサイズ・色も変更可能。
- **テキスト回転**: テキストを自由な角度に回転（🆕）。
- **ハイライト**: 重要な箇所をマーカーで強調。
- **編集・移動**: 注釈の移動、リサイズ、削除もスムーズ。

### ⚡ 高度な機能
- **Undo / Redo**: 操作の取り消し・やり直しに対応（`Ctrl+Z`, `Ctrl+Y`）。
- **右クリックメニュー**: コンテキストメニューから素早くアクションを実行。
- **ズーム機能**: 拡大・縮小で細部まで確認。
- **エクスポート**:
    - **PDF保存**: 編集内容をPDFとしてダウンロード。
    - **画像出力**: ページをPNG画像や、全ページ一括ZIPとして保存。

## 🎮 使い方 (How to Use)

### 基本操作
1. **ファイルを開く**: 中央の「ファイルを選択」ボタン、またはファイルをドラッグ＆ドロップ。
2. **ページ移動**: サイドバーのサムネイルをクリック、またはカーソルキー(`↑` `↓`)。
3. **保存**: 右上の「ダウンロード」ボタン（`Ctrl+S`）で編集後のPDFを保存。

### 編集ツール
- **テキスト (T)**: ツールバーの「テキスト」を選択し、好きな場所に入力。
- **ハイライト (H)**: ツールバーの「マーカー」を選択し、強調したい範囲をドラッグ。
- **要素の編集**: 配置したテキストやハイライトをクリックして選択。
    - **移動**: ドラッグ
    - **回転**: 上部のハンドルをドラッグ（テキストのみ）
    - **削除**: `Delete`キー

## ⌨️ キーボードショートカット

| 区分 | 操作 | Windows | Mac |
|------|------|---------|-----|
| **ファイル** | 開く | `Ctrl + O` | `Cmd + O` |
| | 保存 | `Ctrl + S` | `Cmd + S` |
| **編集** | Undo (元に戻す) | `Ctrl + Z` | `Cmd + Z` |
| | Redo (やり直し) | `Ctrl + Y` / `Shift+Z` | `Cmd + Y` / `Shift+Z` |
| **選択** | 全ページ選択 | `Ctrl + A` | `Cmd + A` |
| **ページ** | 削除 | `Ctrl + D` | `Cmd + D` |
| | 前のページ | `↑` | `↑` |
| | 次のページ | `↓` | `↓` |
